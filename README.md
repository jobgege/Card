# Card
这是大一的c语言程序设计作业，基于原神和碧蓝航线实现的抽卡模拟器（80抽保底），来试试你是欧皇还是非囧

运行软件：Visaul Studio 2022
主程序入口：ConsoleApplication1.sln
测试用户账号：1 密码：空
